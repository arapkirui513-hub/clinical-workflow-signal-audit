# Responsible AI and Data Handling

## Data

This project uses synthetic ICU workflow data only.

It does not include:

- Real patient records
- Protected health information
- Hospital-identifiable data
- Real clinician identifiers
- Real operational logs

## Product positioning

Clinical Workflow Signal Audit is a workflow intelligence demo.

It does not diagnose patients or replace clinical judgment.

## Safety principles

- Keep clinicians in control.
- Allow human override.
- Show why a signal was routed.
- Suppress scoring when data quality is poor.
- Separate workflow risk from diagnosis.
- Audit routing decisions.
- Use synthetic or governed data for development.
